Raspberry Pi Live Streaming System
📌 Overview
RaspberryPi Live Streaming System is an IoT-based project built using Raspberry Pi and Python to enable real-time video streaming, sensor monitoring, and remote device control over a network. This project is suitable for surveillance, smart monitoring, and edge-computing experiments.
The system demonstrates practical use of embedded Linux, networking, and Python-based automation—ideal for learning and portfolio showcasing.

✨ Features
📷 Live Video Streaming using Raspberry Pi Camera / USB Camera
🌐 Real-time access over network (LAN / VPN)
🧠 Python-based backend for device control
🔌 GPIO integration for sensors and actuators
⚡ Lightweight and efficient for edge devices

🛠️ Hardware Requirements
Raspberry Pi (Zero 2 W)
Raspberry Pi Camera Module or USB Webcam
MicroSD Card (16 GB or higher recommended)
Power Supply

Sensors: (RCWL-0516, Temperature, etc.)
